# Elsmore

A fabulous fellow who will fetch entire websites for you and rewrites them to your wishes.

![Elsmore](elsmore.png)

Might need a bit of a push to get him going.

## To install
```sh
gem install elsmore
```

## To fetch website
```sh
elsmore go fetch [website_url] [--debug]
```

## To serve
```sh
elsmore serve [website_url]
```
